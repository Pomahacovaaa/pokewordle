## Made using
* HTML
* CSS
* JavaScript
### and
* mixkit
* DaFont
* PokemonDatabase
## Features
* guessing a pokemon (out of my "database" in pokemon.js)
* sound effects (background music...)
* an option to visit the pokedex after finishing the game
## How to play
1. guess a pokemon
2. based on the colors guess another one <br>
🟩<b>green</b> - correct<br>
🟨<b>yellow</b> - different placement<br>
⬛<b>grey</b> - incorrect letter<br>
